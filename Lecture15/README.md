# 📚 Lecture 15 - React Routing, Store & Custom Hooks

## 🚀 Topics Covered

- Learned **React Routing** using `react-router-dom`.
- Understood how to create **nested routes** and navigate between pages.
- Learned about **Store (State Management)** using Zustand.
- Created and managed **global state** inside a store.
- Built a **Custom Hook** to reuse logic across multiple components.
- Understood why custom hooks improve **code reusability** and keep components clean.

## 📖 Key Learnings

- React Router helps create multiple pages in a React application.
- A Store allows sharing state between components without prop drilling.
- Custom Hooks make repeated logic reusable and easier to maintain.
